Abstract

InsulinX provides a simple solution to calculate multiple index from a selective list based on the most used indexes in medical studies related to diabetes. This application is intended for both specialized persons and non-specialized. For non-specialized patients, we provide a additional page "Read me" where you can find informations about diabetes, how those indices are evaluated and how the result are computed.

The application was built to be used without an account, providing all features needed in order to calculate and see the results based on the resulted values and inserted data. However, the application provides additional features for the ones that have an account created.

Features

For a person without an account, the application is limited to a specific list of features, including: Register, access to Read Me page to find more information about diabetes, access to Contact Me page, useful for contacting the developer team regarding bugs or implementation of additional features. But the most important feature is the possibility to select a list of desired indexes and to calculate them using minimal information. It also provides a result page, where resides the graphical interpretation of the inserted data and a table with each indices calculated and interpreted.

Benefits of an account

A person who chose to create an account has access to everything that the application has to offer. Besides the features listed in the previous section, a logged user has access to a settings page, from where he can manipulate his account and stored data, to the history page, where one can see all indices calculated in the past, sort and filter the history page, delete indices and also download an Excel which contains all the indices listed on the History page. Also, on the Result page, the logged user has the possibility to download an excel with the inserted data and the calculated values or to generate a PDF report with the charts and the corresponding interpretation.Any user can create as many accounts as he wants, there is no restriction regarding the number of accounts and no additional pay is requested for creating one.

Note

Although the application provides an estimation regarding the health of the patient, you should not take the result generated on the application as grounded. One should consult a doctor right away in case of the presence of symptoms specific to diabetes or pre-diabetes. Also, try to consult a doctor if the results of the application are indicating a potential problem regarding your personal health.
