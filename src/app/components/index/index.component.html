Diabetes classification

Using the latest techniques and algorithms from Machine Learning industry, the application can detect if a person has or not diabetes.

The model has an accuracy of 95% in detecting diabetes, in any stage, based on simple data obtained from medical analysis, easy to obtain by anyone. Using this powerful tool, one can know if he is suspect of diabetes and maybe can prevent it.

However, in case of symptoms, one should consult a specialist, whatever the prediction of the model. It is indicated to consult a medic also when the model predicts that someone has diabetes, to exclude any possible risk or complication.

Diabetes evolution

Utilizing regression techniques, one can check how the disease will evolve in one year. As the previous model, it is using the latest information about regression problems.

Using easy to obtain data, the model can predict a quantitative measurement which is used to measure how the disease will evolve after a year from the current day. The less the value obtained, the better is your current lifestyle or treatment.

The predicted value vary from 25 to and 346, and a value bigger or equal to 250 means that you should do a massive change in your lifestyle. A value between 150 and 250 signify that one should keep an eye on the current situation. A value below 150 means that the disease is held under control correctly.
